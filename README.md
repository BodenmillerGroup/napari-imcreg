## Installation

You can install `napari-imcreg` via [pip]:
```shell
pip install napari-imcreg
```

As napari is in active development, the use of virtual environments is strongly recommended.

## Usage

To start `napari-imcreg` from command line:

```shell
napari-imcreg
```
